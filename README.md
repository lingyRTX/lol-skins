## ✨ Features

### Core Functionality
- **🎨 Custom Skin Application**: Apply any League of Legends skin to any champion
- **⭐ Favorites System**: Save and organize your favorite skins for quick access
- **🔍 Champion Search**: Quickly find champions with intelligent search functionality
- **🌈 Chroma Support**: Full support for skin chromas with preview functionality
- **📱 Real-time Champion Detection**: Automatically detect champion selection in-game
- **🎯 Skin Variants**: Support for special skin variants (K/DA, Risen Legend, etc.)

### Advanced Features
- **📁 Custom Skin Upload**: Import and manage your own custom skin files (.zip, .fantome)
- **🔄 Auto-Updates**: Automatic skin database and metadata updates
- **🌍 Multi-language Support**: Available in English and Russian
- **🎮 League Client Integration**: Seamless integration with League of Legends client
- **💾 Local Storage**: All favorites and settings stored locally for privacy

### User Interface
- **🎨 Modern Design**: Clean, League of Legends-themed interface
- **📱 Responsive Layout**: Optimized for different screen sizes
- **🔍 Advanced Filtering**: Filter skins by rarity, champion, and more
- **⚡ Fast Performance**: Optimized for smooth user experience

## 📦 Installation

### Download and Install
1. Download the latest release from the [Releases page](https://github.com/lingyRTX/lol-skins/releases)
2. Run `l-skins-1.0.2-setup.exe` as Administrator
3. Follow the installation wizard
4. Launch L Skins from Desktop or Start Menu

### First Launch Setup
1. **License Activation**: Enter your license key when prompted
2. **League Path Detection**: The app will automatically detect your League installation
   - If not found automatically, manually select your League of Legends folder
3. **Initial Download**: Wait for skin database and assets to download (may take 5-10 minutes)

## 🚀 Quick Start Guide

### Basic Usage
1. **Launch the Application**
   ```
   Double-click L Skins icon or run from Start Menu
   ```

2. **Select a Champion**
   - Use the sidebar to browse champions alphabetically
   - Use the search bar to quickly find specific champions
   - Click on any champion to view their available skins

3. **Apply a Skin**
   - Browse through available skins for your selected champion
   - Click on any skin to apply it immediately
   - Use chroma circles to apply different color variants

4. **Manage Favorites**
   - Click the heart icon on any skin to add it to favorites
   - Access favorites through the header button
   - Remove favorites by clicking the heart icon again

### Advanced Features

#### Custom Skin Upload
```
1. Click the "Custom Skins" button in the header
2. Click "Upload Skins" button
3. Select your .zip or .fantome files
4. Files will be automatically processed and added to your collection
```

#### Champion Select Detection
```
1. Click the target icon in the header to enable detection
2. Start a League of Legends match
3. L Skins will automatically detect your champion selection
4. Relevant skins will be displayed automatically
```

#### Skin Database Updates
```
1. Go to Settings (gear icon in header)
2. Click "Update Skin Metadata" for quick updates
3. Click "Update Skins Database" for full database refresh
```
## 🔧 Troubleshooting

### Common Issues

#### "League of Legends not found"
- Ensure League is installed in the default location
- Manually select League folder in settings
- Run L Skins as Administrator

#### "License activation failed"
- Check your internet connection
- Verify license key format (XXXX-XXXX-XXXX-XXXX)
- Contact support if issues persist

#### "Skin not applying in-game"
- Ensure League client is closed when applying skins
- Restart League client after applying skins
- Check that the skin file exists and is not corrupted

#### "Champion detection not working"
- Enable champion select detection in header
- Ensure League client is running and logged in
- Check that L Skins has necessary permissions

### Performance Optimization
- Close unnecessary applications while using L Skins
- Regularly clean temporary files from `%APPDATA%/l-skins/temp/`
- Update to the latest version for performance improvements
